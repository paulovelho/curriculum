Another project that I was responsible during my time at Alpar was an Ionic mobile application.
This project was for another big company, that already had a web-based application for managing HR related requests, such as vacations and sick days. The company ordered a low-budget simple app for the managers to get notifications on their mobile when an employee requested anything.
The solution was to create an Ionic application that loads the web-based responsive website in a WebView and manages through Firebase the devices and authentication information for each user, therefore sending the push notifications when necessary.
