Projects

Building the banking application was the biggest task I did during my period at Savedo.
Since the beginning, we decide to use the modern technologies and languages in order to build a platform that could grow healthy and with high-end features.
We developed a full front-end application to interact with a back-end API.
The application was built with Angular 1.0 and Ionic built over Typescript. The whole project was full covered with tests provided by mocha framework. The communication with the main servers were following all the jsonapi patterns.
API was built in Ruby on Rails. Although I was not responsible for the inner engines of the API (backend), it was not rare for me to put my hands on the server code to fix/build something.
